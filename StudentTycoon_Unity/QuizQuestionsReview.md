# 문제/정답 검토용

이 파일은 현재 씬에 저장된 문제 데이터를 정리한 것입니다.
OX 문제의 정답은 `O` 또는 `X`로 적었습니다.
입력 문제는 플레이어가 입력해야 하는 정답을 적었습니다.

## 내가 봤을 때 확인하면 좋은 후보

- 과학 `물은 0도에서 언다 ?`: 초등 수준에서는 O로 괜찮지만, 정확히는 일반적인 조건/표준기압 기준이라고 붙이면 더 좋습니다.
- 과학 `물이 끓는 온도는 섭씨 몇 도 ?`: 이것도 일반적인 조건/표준기압 기준이면 100이 맞습니다.
- 과학 `산소의 원소 기호는 ?`: 정답이 `O`라서 OX 버튼의 O와 헷갈릴 수 있습니다. 입력 문제라 기능상 문제는 없습니다.
- 사회 `우리나라 광복절은 몇 월 15일 ?`: 정답을 `8`로 해두어서 `8월`이라고 입력하면 오답 처리됩니다.
- 프로그래밍 `명령을 순서대로 적은 것은 ?`: 정답을 `코드`로 해뒀는데, 학생이 `프로그램`이라고 생각할 수도 있습니다.
